# 🧪 비디오 편집기 기능 테스트 가이드

## ✅ 기능 1: 캡처에 텍스트/워터마크 추가

### 테스트 준비
1. `Ctrl + Shift + R`로 브라우저 강제 새로고침
2. 비디오 파일 로드 또는 URL 입력

### 테스트 시나리오

#### 시나리오 1-1: 텍스트 없이 캡처 (기본 동작)
**단계:**
1. 비디오를 재생하고 원하는 지점에서 일시정지
2. "텍스트 내용" 필드를 **비워둠**
3. `📸 PNG 캡처` 버튼 클릭

**예상 결과:**
- ✅ 텍스트 없이 깔끔한 이미지 캡처
- ✅ 2배 고해상도로 선명하게 저장
- ✅ 파일명: `capture_YYYY-MM-DDTHH-MM-SS.png`

#### 시나리오 1-2: 텍스트 추가 캡처
**단계:**
1. "텍스트 내용"에 `테스트 워터마크` 입력
2. 폰트 크기: `보통 (36px)` 선택
3. 텍스트 색상: `흰색` 선택
4. 텍스트 위치: `하단` 선택
5. `📸 PNG 캡처` 버튼 클릭

**예상 결과:**
- ✅ 이미지 하단 중앙에 "테스트 워터마크" 표시
- ✅ 반투명 검정 배경으로 텍스트 가독성 좋음
- ✅ 텍스트 외곽선으로 선명하게 보임

#### 시나리오 1-3: 다양한 옵션 테스트
**단계:**
1. 텍스트: `© 2024 My Channel`
2. 폰트 크기: `크게 (48px)`
3. 색상: `노랑` 선택
4. 위치: `상단` 선택
5. JPG 캡처 버튼 클릭

**예상 결과:**
- ✅ 이미지 상단에 노란색 큰 텍스트 표시
- ✅ JPG 형식으로 저장 (최대 품질)

## ✅ 기능 2: 타임스탬프 자동 표시

### 테스트 준비
1. `Ctrl + Shift + R`로 브라우저 강제 새로고침
2. 비디오 파일 로드
3. 비디오를 **15초** 지점으로 이동

### 테스트 시나리오

#### 시나리오 2-1: 타임스탬프만 표시
**단계:**
1. "⏱️ 타임스탬프 자동 표시" 체크박스 **체크**
2. "텍스트 내용"은 **비워둠**
3. 비디오를 정확히 `00:00:15` 지점에 위치
4. `📸 PNG 캡처` 버튼 클릭

**예상 결과:**
- ✅ 이미지 우측 하단에 `00:00:15.XX` 형식 표시
- ✅ 검정 배경 박스에 흰색 텍스트
- ✅ Courier New 모노스페이스 폰트

#### 시나리오 2-2: 타임스탬프 + 커스텀 텍스트 함께
**단계:**
1. "텍스트 내용"에 `중요 장면` 입력
2. 텍스트 위치: `하단` 선택
3. "타임스탬프 자동 표시" **체크**
4. 비디오를 `00:01:30` 지점으로 이동
5. `📸 PNG 캡처` 버튼 클릭

**예상 결과:**
- ✅ 하단 중앙: "중요 장면" 텍스트
- ✅ 우측 하단: `00:01:30.XX` 타임스탬프
- ✅ 두 텍스트가 겹치지 않고 깔끔하게 배치

#### 시나리오 2-3: 타임스탬프 비활성화
**단계:**
1. "타임스탬프 자동 표시" 체크박스 **해제**
2. `📸 PNG 캡처` 버튼 클릭

**예상 결과:**
- ✅ 타임스탬프가 표시되지 않음
- ✅ 일반 캡처만 수행

#### 시나리오 2-4: 다양한 시간대 테스트
**단계:**
1. "타임스탬프 자동 표시" **체크**
2. 다음 시간대에서 각각 캡처:
   - `00:00:05` → 예상: `00:00:05.XX`
   - `00:01:23` → 예상: `00:01:23.XX`
   - `01:05:45` → 예상: `01:05:45.XX` (1시간 이상)

**예상 결과:**
- ✅ 모든 시간대가 올바른 형식으로 표시
- ✅ 밀리초(.XX)도 정확하게 표시

## 🔄 복합 테스트

### 복합 시나리오 1: 모든 기능 동시 사용
**단계:**
1. 텍스트: `🎬 영화 제목 - 명장면`
2. 폰트 크기: `매우 크게 (64px)`
3. 색상: `흰색`
4. 위치: `상단`
5. 타임스탬프: **체크**
6. 비디오 위치: `00:02:15`
7. PNG 캡처

**예상 결과:**
- ✅ 상단: 큰 흰색 텍스트 "🎬 영화 제목 - 명장면"
- ✅ 우측 하단: `00:02:15.XX` 타임스탬프
- ✅ 고해상도 선명한 이미지

### 복합 시나리오 2: 폴더 저장 + 텍스트
**단계:**
1. `📂 폴더 선택` 버튼으로 저장 폴더 지정
2. 텍스트: `테스트 이미지`
3. 타임스탬프: **체크**
4. PNG 캡처

**예상 결과:**
- ✅ 지정한 폴더에 이미지 저장됨
- ✅ 상태 메시지: "파일명이(가) 폴더명 폴더에 저장되었습니다"

## 📋 체크리스트
